# こんにちは✏️ halftone gen (alpha)です

## About
* ハーフトーンパターンのようなものをイラレでつくるスクリプトです
* 個人制作のalpha版です。問題や不具合があればお知らせ下さい
* 3D版はこちら → [3d halftone gen](https://github.com/ykwe/3d_halftone_gen/)

## 使い方
* [halftone_gen.jsx](halftone_gen.jsx) をダウンロードし、イラレのスクリプトとして開いてください

## 設定項目

<img width="400" height="744" alt="設定画面" src="https://github.com/user-attachments/assets/c5d805fc-0b46-4e0e-8287-d5573a550b5e" />

#### ドット個数
* 縦と横のドットの個数を指定します
#### シェイプ設定
* ドットの形（シェイプ）や位置を指定します
* シェイプは丸 / 四角から選べます
#### パターン
* ドットの並び方（パターン）を指定します
* 千鳥配置にすると、互い違いにずれた並びになります
* 傾斜角度を変更すると、ドットの並びが斜めになります
#### ランダム
* ドットの位置、サイズ、回転にそれぞれランダムな揺らぎを加えます
* 指定した範囲内で各パラメータの数値が増減されます
#### グラデーション
* ドットのサイズを一定方向で徐々に変化させます
* グラデーション方向の角度と、始点・終点の濃度を指定します
* シェイプを変形させる際の方法（通常 / 幅のみ変形 / 高さのみ変形）が選べます
#### 設定値を出力
* 設定項目をテキストとして生成します
* 一度出力したドットパターンを再現したいときに便利です
