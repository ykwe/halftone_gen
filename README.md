## 設定項目

<img width="400" height="744" alt="設定画面" src="https://github.com/user-attachments/assets/c5d805fc-0b46-4e0e-8287-d5573a550b5e" />

#### ドット個数
* 縦と横のドットの個数を指定します
#### シェイプ設定
* ドットの形（シェイプ）や位置を指定します
* シェイプは丸 / 四角形から選べます
#### パターン
* ドットの並び方（パターン）を指定します
* 千鳥配置にすると、互い違いにずれた並びになります
#### ランダム
* 位置、シェイプのサイズと回転に揺らぎを加えます
* 指定した数値の範囲内でノイズが追加されます
#### グラデーション
* シェイプのサイズを、一定の範囲内で変化させます
* グラデーションの方向の角度、始点と終点の濃度を指定します
#### 設定値を出力
* 設定項目をテキストとしてパターンと一緒に出力します
